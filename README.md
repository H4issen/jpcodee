# JP Code - Portafolio Profesional

Portafolio profesional — IT Support Technician, Web Developer y Data Analyst.

## Contenido

- `index.html` — Página principal con video de presentación, sección de proyectos (tarjetas con enlaces a repositorios y demos), y footer con información de contacto.
- `servicios.html` — Página de servicios con slider automático de imágenes.
- `css/style.css` — Estilos globales: layout responsive, animaciones, loader de precarga, slider, header fijo con blur, footer y tipografía Poppins.
- `js/index.js` — Configuración de ScrollReveal para animaciones de entrada al hacer scroll.
- `js/preloader.js` — Script que precarga todas las imágenes del sitio y oculta el loader cuando están listas.
- `js/scrollreveal.js` — Librería ScrollReveal (v4.0.9).
- `images/` — Imágenes de proyectos, slider, logos.
- `videos/` — Video de fondo para el hero de la página principal.
- `documents/` — CV en PDF.

## Tecnologías utilizadas

- HTML5
- CSS3 (responsive design, animaciones con keyframes, Flexbox, media queries)
- JavaScript (ScrollReveal, precarga de imágenes)
- Bootstrap Icons / Boxicons

## Uso

Abrir `index.html` en un navegador. No requiere servidor ni dependencias adicionales.
